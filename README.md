# R182 Commander

Scenario-driven aircraft command training built with Next.js, TypeScript, Tailwind CSS, and Prisma.

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Database

Copy `.env.example` to `.env` and set `DATABASE_URL` before running Prisma commands.

```bash
npm run db:validate
npm run db:generate
```
